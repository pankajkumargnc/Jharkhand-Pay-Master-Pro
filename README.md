## 🚀 Installation & Setup

### Prerequisites
- [Node.js](https://nodejs.org/) v18 or higher
- npm v9 or higher

### Quick Start

```bash
# Clone repository
git clone https://github.com/pankajkumargnc/jharkhand-pay-master-pro.git
cd jharkhand-pay-master-pro

# Install dependencies
npm install

# Start development server
npm run dev
# Opens at: http://localhost:3000
```

### Build for Production

```bash
npm run build
npm run preview
```

### 🌐 Deploy Free on Vercel

```bash
npm install -g vercel
vercel
# Your app is live in 60 seconds!
```

## 🏦 Key Formulas

### Gratuity
```
Gratuity = (Last Basic + DA) × 15 × Qualifying Service Years ÷ 26
Maximum: ₹20,00,000
```

### NPS
```
Employee: 10% of (Basic + DA)
Government: 14% of (Basic + DA)
Total Monthly: 24% of (Basic + DA)
```

### MACP Upgrades
```
10 years → 1st MACP (Level + 1)
20 years → 2nd MACP (Level + 2)
30 years → 3rd MACP (Level + 3)
```

## 💾 Data Backup Guide

> ⚠️ **Critical:** All data is stored in browser localStorage. Clearing browser = all profiles lost!

**Backup methods:**
1. **Settings** tab → **Backup All Data** → Excel file saved
2. **All Profiles** tab → **Export All** → Excel backup
3. Orange reminder banner appears **automatically after 7 days**
4. Reset button **auto-exports** before clearing

**Restore:** All Profiles → Import Excel → Select backup file
